# ⚽ ProTicket: The Future of Sports Ticketing

**Developed using Google Antigravity (Agent-first IDE)** *An advanced Full-Stack application integrating AI-driven recommendations with real-time database management.*

---

## 🌟 Overview
ProTicket is not just a ticketing platform; it's a smart assistant for sports enthusiasts. Built with a focus on seamless user experience and modern "Anti-Gravity" design principles, the app allows users to search, sort, and book tickets for major football matches with the help of **Astro**, an integrated AI agent.

## 🚀 Key Features
- **🤖 Astro AI Assistant:** A Gemini-powered sidekick that provides personalized match recommendations based on user mood and preferences.
- **⚡ Real-time Search & Filter:** Instantly filter matches by team name using an optimized search bar.
- **📊 Dynamic Price Sorting:** Sort available tickets from lowest to highest price with a single click.
- **🌐 Cloud Integration:** Fully connected to a **MongoDB Atlas** cloud database for real-time data persistence.
- **📱 Responsive "Electric Cyan" UI:** A futuristic dark-mode design built for both desktop and mobile using Tailwind CSS and Framer Motion for floating animations.

## 🛠️ Tech Stack
- **Frontend:** React.js (Vite), Tailwind CSS, Framer Motion (for high-fidelity animations).
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB Atlas (NoSQL).
- **AI Integration:** Google Gemini 1.5 Flash API.
- **Environment:** Developed in an Agent-First workflow within the Antigravity IDE.

## ⚙️ Installation & Setup
1. **Clone the project** and open it in your environment.
2. **Backend Setup:**
   - Navigate to `/server`.
   - Create a `.env` file with your `MONGO_URI` and `GEMINI_API_KEY`.
   - Run `node seed.js` to populate the database.
   - Start the server using `node index.js`.
3. **Frontend Setup:**
   - Navigate to `/client`.
   - Run `npm install` (if needed) and `npm run dev`.
